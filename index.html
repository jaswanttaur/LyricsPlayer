LyricsPlayer

mp3 lyrics player (java)

  1. open Lyrics Player in netbeans
  2. there is a compiled project in Lyrics Player -> dist or you can open the source of project in the netbeans
  3. before starting the project install jdk, jmf & Arial Unicode MS font.
  4. then open the project, change source and recompile according to your requirements.
  5. import the database file lyrics.sql by making lyrics database in the database
  6. For any other queries email me at bsinghs.007@gmail.com
